5. Sesión 4: Experimento

Experimento

En esta sesión vamos a realizar un experimento para analizar el movimiento de un objeto a partir de datos reales.

Nos convertiremos en científicas y científicos y recogeremos información para estudiar cómo se mueve un objeto.

 
Experimento
Vamos a medir el movimiento de un objeto en un espacio real (por ejemplo, en el patio o en el aula).

Puedes utilizar:

Una pelota
Un coche de juguete
Una persona caminando
 
¿Qué vamos a medir?
Para analizar el movimiento necesitamos recoger dos datos:

Distancia recorrida (en metros)
Tiempo empleado (en segundos)
 
Recogida de datos
Crea una tabla como esta:

Tiempo (s):
Distancia (m):

Anota los datos a medida que el objeto se mueve.

 
Organización del trabajo
Trabajarás en grupo y cada persona tendrá un rol:

Medir el tiempo
Medir la distancia
Registrar los datos
 
Puedes registrar los datos utilizando:

Cuaderno
Hoja de cálculo (Excel o similar)
Después, estos datos se utilizarán en la siguiente sesión.

 
Objetivos de la actividad:
Recoger datos reales de movimiento
Trabajar en equipo
Preparar la información para su análisis
 
Importante:
Intenta que las medidas sean lo más precisas posible para que el análisis posterior sea correcto.

 
Cierre:
En la siguiente sesión analizaremos estos datos utilizando herramientas digitales para comprender mejor el movimiento.
